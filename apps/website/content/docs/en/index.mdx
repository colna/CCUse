# CCUse Docs

This page verifies the documentation pipeline before the full guide content lands.

## Local Proxy Quick Start

Point any OpenAI-compatible client at the local endpoint:

### Client Endpoint

```ts
const baseUrl = "http://127.0.0.1:8787";
const apiKey = "sk-local-example";
```

Then add providers in CCUse and keep your client configuration unchanged.

Read [Getting Started](/en/docs/getting-started) for the full installation and client setup flow.

Read [Switching Strategies](/en/docs/switching-strategies) to choose between priority, fastest, cost-first, load-balance, and smart routing.

Read [Monitoring and Alerts](/en/docs/monitoring-alerts) to interpret dashboard metrics and inspect switch history.
