# CCUse 文档

这个页面用于验证完整用户手册落地前的 MDX 文档管线。

## 本地代理快速开始

把任意 OpenAI 兼容客户端指向本地端点：

### 客户端端点

```ts
const baseUrl = "http://127.0.0.1:8787";
const apiKey = "sk-local-example";
```

然后在 CCUse 中添加供应商，客户端配置无需跟随上游供应商变化。

完整安装与客户端配置流程见[入门指南](/zh/docs/getting-started)。

选择优先级、最快响应、成本优先、负载均衡或智能路由前，请阅读[切换策略](/zh/docs/switching-strategies)。

查看仪表盘指标与切换历史时，请阅读[监控与告警](/zh/docs/monitoring-alerts)。

遇到端口冲突、WebView2、公证或 Windows 安全提示时，请阅读[FAQ 与故障排查](/zh/docs/faq-troubleshooting)。
